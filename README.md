# BT21 Talk
2021년 6월, 개인프로젝트

### 소개
HTML과 CSS를 배운 후 제가 좋아하는 BT21 캐릭터를 활용하여 메신저 화면을 마크업했습니다.

### 구현 기능
- HTML의 form, list, a 태그 등을 사용해 각 페이지를 마크업
- CSS의 flex box, position, z-index, transition, media query 등을 사용해 스타일링
